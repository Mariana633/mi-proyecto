# Mi Proyecto

Este proyecto es una práctica de Git y GitHub donde se trabaja con
repositorios locales y remotos, manejo de ramas y Pull Requests.

## Funcionalidades
- Página HTML básica
- Barra de navegación simulada
- Control de versiones con Git

## Ramas utilizadas
- main: rama principal del proyecto
- feature/navbar: desarrollo de la barra de navegación
